# unity-ui

Generate UGUI and UI Toolkit code with responsive layouts, data binding, and animation.

## When to use

Activate when the user mentions:
- Creating UI panels, menus, HUD, dialogs
- UGUI or UI Toolkit / UI Builder
- Responsive layout for multiple resolutions
- UI animations and transitions
- Data binding for UI elements

## Capabilities

1. **Panel generation** — Complete UI panel code with open/close animations
2. **Layout systems** — Responsive layouts for mobile/tablet/desktop
3. **UI architecture** — MVC/MVP patterns for game UI
4. **Component library** — Reusable UI components (buttons, lists, popups)
5. **UI Toolkit migration** — UGUI to UI Toolkit conversion guidance
