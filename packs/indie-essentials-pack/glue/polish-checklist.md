# Indie Polish Checklist

## Game Feel ("Juice")

### Camera
- [ ] Screen shake on impact
- [ ] Smooth follow with lookahead
- [ ] Dynamic FOV on speed change
- [ ] Subtle idle movement

### Input
- [ ] Coyote time (grace period for jumps)
- [ ] Input buffering (early presses stored)
- [ ] Forgiveness frames (lenient hit detection)

### Visual
- [ ] Particle effects on key actions
- [ ] Squash and stretch on characters
- [ ] Flash/freeze on critical hits
- [ ] Trails on fast movement

### Audio
- [ ] Layered music system
- [ ] Pitch variation on repeated SFX
- [ ] Audio ducking (music lowers on SFX)
- [ ] Reverb zones for atmosphere

## Platform Requirements

### Steam
- [ ] Achievements implemented
- [ ] Cloud save enabled
- [ ] Trading cards (if eligible)
- [ ] Controller support

### Mobile (iOS/Android)
- [ ] Touch controls feel responsive
- [ ] Battery drain tested
- [ ] App store screenshots prepared
- [ ] Privacy policy included

### itch.io
- [ ] Cover image (630x500)
- [ ] GIF trailer embedded
- [ ] Price tier decided
- [ ] Demo build available

## Marketing Assets

### Store Page
- [ ] Elevator pitch (1 sentence)
- [ ] Short description (2-3 paragraphs)
- [ ] 5+ screenshots
- [ ] Trailer (30-60 seconds)

### Social Media
- [ ] Twitter/X account
- [ ] Devlog schedule (weekly/bi-weekly)
- [ ] GIFs for social sharing
- [ ] Discord server (optional)

## Final QA

- [ ] Played through entire game
- [ ] Tested on target minimum hardware
- [ ] No console errors
- [ ] Build size optimized
- [ ] Loading times acceptable
- [ ] No soft-locks possible
